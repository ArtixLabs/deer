## Installation

```sh
cp deer.zsh ~/.zsh/ && echo -e "source ~/.zsh/deer.zsh" >> ~/.zshrc
```

## Uninstallation

```sh
rm -rf ~/.local/share/deer
```

## Example usage

```sh
deerplug "zsh-users/zsh-autosuggestions"
deerplug "zsh-users/zsh-syntax-highlighting"
```

## Advanced Commands

```
deerupdate
```
